# Data Drop

An AI career program marketing website for "India's Most Practical AI Career Program."

## Stack

- **Framework:** React 19 + TanStack Start (TanStack Router + Vite)
- **UI:** shadcn/ui components, Tailwind CSS v4
- **Runtime:** Bun
- **Language:** TypeScript

## Pages

- `/` — Home (hero, features, curriculum preview, pricing, testimonials)
- `/curriculum` — Full curriculum breakdown
- `/projects` — Real-world project showcase
- `/pricing` — Pricing plans
- `/faq` — Frequently asked questions
- `/about` — About the program
- `/contact` — Contact form
- `/privacy`, `/terms`, `/refund`, `/shipping` — Legal pages

## Running locally

```bash
bun install
bun run dev      # starts dev server on port 5000
bun run build    # production build
```
